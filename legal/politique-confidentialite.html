Politique de confidentialité de Politaria

La présente politique de confidentialité décrit la manière dont Politaria (le « Service ») collecte, utilise et protège vos données. En utilisant Politaria, vous acceptez les pratiques décrites ci-dessous.

1. Données collectées

Politaria collecte uniquement les informations nécessaires au fonctionnement du jeu :

  • Données de connexion : adresse e-mail lorsque vous créez un compte via Google.
  • Données de jeu : systèmes stellaires créés, progression, événements et sauvegardes.
  • Données invité : stockées localement sur votre appareil (aucune transmission vers nos serveurs).
  • Données techniques : appareil, navigateur, résolution d’écran, performances.

Aucune donnée sensible (âge, identité réelle complète, adresse postale, etc.) n’est collectée.

2. Utilisation de vos données

Les données collectées ont pour finalité :

  • assurer le bon fonctionnement de Politaria ;
  • permettre la sauvegarde et la synchronisation de vos systèmes entre plusieurs appareils ;
  • améliorer le service, corriger les bugs et optimiser les performances ;
  • prévenir les abus, fraudes ou usages anormaux.

Vos données ne sont jamais revendues, données ou louées à des tiers.

3. Mode invité

En mode invité, vos données ne quittent jamais votre appareil. Elles sont stockées uniquement via le stockage local du navigateur. Si vous effacez vos données locales, votre progression invitée sera définitivement perdue.

4. Partage de données

Nous pouvons partager certaines données avec :

  • Google OAuth : uniquement pour l’authentification ;
  • Services d’hébergement : pour faire fonctionner le jeu et sauvegarder vos mondes ;
  • Outils d’analyse anonymisés (si activés) : pour comprendre comment améliorer le jeu.

Aucune donnée n’est partagée sans nécessité technique.

5. Sécurité des données

Politaria met en place des mesures techniques et organisationnelles pour protéger vos données contre tout accès non autorisé, modification ou destruction. Cependant, aucun service en ligne n’est totalement exempt de risques.

6. Vos droits

Vous pouvez, à tout moment :

  • demander un accès à vos données ;
  • corriger les informations associées à votre compte ;
  • supprimer définitivement votre compte et vos mondes sauvegardés ;
  • retirer votre consentement à l’utilisation du service.

Pour exercer ces droits, contactez-nous à l’adresse suivante :

support@politaria.eu

7. Cookies et stockage local

Politaria utilise uniquement des cookies ou stockages locaux strictement nécessaires au fonctionnement du jeu :

  • Mode invité : progression stockée localement ;
  • Session utilisateur : connexion via Google ;
  • Préférences du jeu : affichage, paramètres visuels, mode sombre, etc.

Aucun cookie publicitaire n’est utilisé.

8. Modifications de cette politique

Nous pouvons mettre à jour cette politique lorsque le Service évolue. La version la plus récente sera toujours disponible dans le jeu, via la section légale.

9. Contact

Si vous avez des questions concernant la confidentialité ou l’utilisation de vos données, contactez-nous à : support@politaria.eu
